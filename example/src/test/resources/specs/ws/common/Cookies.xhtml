Cookies

To specify cookies the cookies attribute may be used to specify cookies for all cases and the case.cookies attribute to completely replace them for specific case.

Examples

Client has cookie c1 = 111. { "GET": "/mirror/headers", "Authorization": null, "Accept-Language": null, "cookies": {"c1": "111", "c2": "2"} } { "GET": "/mirror/headers", "Authorization": null, "Accept-Language": null, "cookies": {"c2": "22"} }
